## SoundCloud Integration

AURALiS integrates with the SoundCloud platform to enhance the user's music experience.

### Features
- Secure OAuth 2.0 authentication
- Browse and access personal playlists
- Stream tracks from user's SoundCloud library
- View track metadata and artwork

### Technical Implementation
- **Authentication**: OAuth 2.0 with PKCE
- **Redirect URI**: `auralis://soundcloud/callback`
- **Permissions**: Read-only access to user library
- **Data Storage**: All tokens stored locally on-device via iOS Keychain

### API Compliance
This application adheres to the [SoundCloud API Terms of Use](https://developers.soundcloud.com/docs/api/terms-of-use).
